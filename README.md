rainbow.js
====

Site-wide progress indicator that is:  

* Tiny (1KB minified and gzipped), no dependency
* Perfect for single-page/Turbolinks/pjax applications
* Responsive

Check out [demo & usages](http://buunguyen.github.io/rainbow.js).
