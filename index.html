Rainbow.js

Tiny & beautiful progress indicator for JavaScript

Download .zip Download .tar.gz View on GitHub

About

I was working on a single-page AngularJS application which regularly performs tons of small Ajax requests and in need of a way to show progress indicator in a non-obtrusive, non-blocking way (besides being beautiful). After a few hours looking at existing libraries, none of which met my need, I decided to build rainbow.js.

Features

  • Tiny (1KB minified and gzipped)
  • 1 single JS file, no dependency
  • Beautiful and non-intrusive interface
  • Work well with desktop and mobile browsers
  • Non-blocking UI (i.e. no background overlay or modal)

Usages

Simple

Just call rainbow.show() and rainbow.hide(). That's all.

Show Progress Hide Progress 
<script src="src/rainbow.min.js"></script>
<script>
  $('#btnStartSimple').click(function() {
    rainbow.show()
  })
  $('#btnStopSimple').click(function() {
    rainbow.hide()  
  })
</script>

Advanced

Use rainbow.config(opts) to customize rainbow.js. Basically:

  • autoRun (default: true): control whether rainbow.js should automatically increase progress after showing or wait for your command (i.e. invoking rainbow.step().
  • height (default: 10): the height of the canvas upon which rainbow.js draws itself.
  • lineColors: list of mappings from progress position (0 to 1) to color, used to draw the progress line.
  • circleColors: list of mappings from progress position (0 to 1) to color, used to draw the circle.

Note: rainbow.step() can take a number or string value. If string, you can use + or - to change the progress relatively to current position.

Show Progress Increase Hide Progress 
<script src="src/rainbow.min.js"></script>
<script>
  $('#btnStartAdvanced').click(function() {
    rainbow.config({
      autoRun : false, 
      height  : 16,
      lineColors   : {
        '0'   : 'rgba(26,  188, 156, .5)',
        '.3'  : 'rgba(41,  128, 185, .5)',
        '1.0' : 'rgba(231, 76,  60,  .5)'
      },
      circleColors : {
        '0'   : 'rgba(230, 126, 34, .9)',
        '1.0' : 'rgba(241, 196, 15, .9)'
      }
    })
    rainbow.show()
  })
  $('#btnIncreaseAdvanced').click(function() {
    rainbow.step('+.05')  
  })
  $('#btnStopAdvanced').click(function() {
    rainbow.hide()  
  })
</script>

Browsers

rainbow.js works with any browser that supports HTML5 Canvas.
