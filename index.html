Rainbow.js

Tiny & beautiful progress indicator for JavaScript

Download .zip Download .tar.gz View on GitHub

About

I was working on a single-page application which performs tons of Ajax requests and exploring ways to indicate progress in a non-obtrusive and non-blocking manner (besides being good-looking). rainbow.js is the end result.

Features

  • Tiny (1KB minified and gzipped)
  • 1 single JS file, no dependency
  • Beautiful and non-intrusive interface
  • Work well with desktop and mobile browsers
  • Non-blocking UI (i.e. no background overlay or modal)

Usages

Simple

Just call rainbow.show() and rainbow.hide(). That's all.

Show Progress Hide Progress 
<script src="src/rainbow.min.js"></script>
<script>
  $('#btnStartSimple').click(function() {
    rainbow.show()
  })
  $('#btnStopSimple').click(function() {
    rainbow.hide()  
  })
</script>

Advanced

Use rainbow.config(options) to customize rainbow.js.

  • autoRun (default: true): if false, call rainbow.progress() to run.
  • lineThickness (default: 5): the progress bar thickness. 0 to not draw.
  • lineColors: list of gradient color stops used to draw the progress line.
  • circleRadius (default: 5): the leading circle radius. 0 to not draw.
  • circleColors: list of gradient color stops used to draw the leading circle.

Note: rainbow.progress() can take a number or string value. If string, you can use + or - to change the progress relatively to current position. If no argument is specified, return the current progress (0 to 1).

Show Progress Hide Progress 
<script src="src/rainbow.min.js"></script>
<script>
  $('#btnStartAdvanced').click(function() {
    resetToDefaults()
    rainbow.config({
      autoRun       : false, 
      lineThickness : 10,
      lineColors    : {
          '0'       : 'rgba(26,  188, 156, .5)',
          '.3'      : 'rgba(41,  128, 185, .5)',
          '1.0'     : 'rgba(231, 76,  60,  .5)'
      },
      circleRadius  : 0
    })
    rainbow.show();
    (function step() {
      setTimeout(function() {  
        if (rainbow.progress('+.01') < 1) step()
      }, 16)
    })()
  })
  $('#btnStopAdvanced').click(function() {
    rainbow.hide()  
  })
</script>
